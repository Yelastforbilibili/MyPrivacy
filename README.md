快速开始
下载文件，然后配置文件
右键点击文件选择"打开方式"使用本程序加密

使用方法
首次打开文件：设置加密密码

再次打开：输入密码编辑文件

设置中可配置万能密钥

如果忘记了安全界面的密码可输入3930819726 （我的QQ来重新设置加密密码）
或者你可以删除“计算机\HKEY_CURRENT_USER\SOFTWARE\PrivacyApp”这个注册表项目

系统要求
Windows 10/11

[可选]Python 3.6+
